Catatan Pengeluaran adalah aplikasi mobile berbasis Flutter yang digunakan untuk mencatat pengeluaran harian pengguna.  
Aplikasi ini membantu pengguna agar lebih sadar dalam mengelola keuangan dengan mencatat setiap pengeluaran secara sederhana dan mudah digunakan.

Aplikasi dibuat dengan tampilan yang simpel dan nyaman untuk digunakan oleh mahasiswa.

## âš™ï¸ Fitur Aplikasi
Aplikasi Catatan Pengeluaran memiliki fitur sebagai berikut:

-  **Create (Tambah Pengeluaran)**  
  Menambahkan data pengeluaran baru melalui form input.

-  **Read (Menampilkan Data)**  
  Menampilkan daftar pengeluaran yang telah dicatat.

-  **Update (Edit Data)**  
  Mengedit data pengeluaran melalui icon edit.

-  **Delete (Hapus Data)**  
  Menghapus data pengeluaran melalui icon hapus.

-  **Multi Page Navigation**  
  Navigasi antar halaman menggunakan Navigator (HomePage dan FormPage).

-  **Form Input 3 TextField**
  - Judul Pengeluaran  
  - Kategori  
  - Nominal Pengeluaran
